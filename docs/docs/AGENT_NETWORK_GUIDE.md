# CeloHT Agent Network Guide

_Last updated: July 20, 2026_

## Overview

The CeloHT Agent Network connects trusted community members who help expand access to digital financial tools, education, and community support.

Agents serve as local points of assistance while promoting responsible and inclusive technology adoption.

---

# Mission

The mission of the Agent Network is to:

- Improve access to digital financial services.
- Support community members.
- Provide practical education.
- Build trust at the local level.
- Encourage responsible technology use.

---

# Role of an Agent

A CeloHT Agent may support:

- Digital wallet onboarding.
- Community education.
- User assistance.
- Payment guidance.
- Local outreach.
- Feedback collection.

---

# Agent Principles

Agents must follow:

- Transparency
- Integrity
- Respect
- User protection
- Responsible service
- Community values

---

# Agent Responsibilities

Agents should:

- Explain services clearly.
- Help users understand risks.
- Protect user privacy.
- Maintain accurate information.
- Follow CeloHT policies.
- Report important community feedback.

---

# Agent Eligibility

Potential agents should:

- Be trusted members of their community.
- Understand basic digital tools.
- Respect community standards.
- Complete required training.
- Demonstrate responsibility.

---

# Agent Training

Training may include:

## Digital Skills

- Smartphone usage
- Internet safety
- Digital accounts

## Financial Education

- Payments
- Saving concepts
- Responsible financial behavior

## Blockchain Education

- Wallet basics
- Transactions
- Security practices
- Celo ecosystem concepts

## Community Skills

- Communication
- Customer support
- Event participation

---

# Agent Activities

Agents may participate in:

- Community workshops
- User support sessions
- Education campaigns
- Local events
- Feedback programs

---

# Security Responsibilities

Agents must never:

- Request private keys.
- Ask for passwords.
- Access user wallets without permission.
- Misrepresent CeloHT services.

- Community recommendations
- Training programs
- Partnerships
- Regional initiatives

# Monitoring and Improvement

The Agent Network may evaluate:

- Community impact
- User feedback
- Training completion

The Agent Network operates under CeloHT community governance principles.
